Sistema de gerenciamento de tarefas e subtarefas para um usuário

1- Usuário

  • POST /api/app_users – Cria um usuário no sistema, com email e senha.
  • GET /api/app_users – Lista os dados da conta do usuário.
  • PUT /api/app_users/:id – Atualiza informações da conta do usuário.
  • DELETE /api/app_users/:id – Apaga a conta de um usuário.

2- Tarefa

  • POST /api/tarefas – Cria uma tarefa no sistema.
  • GET /api/tarefas – Lista os detalhes da tarefa colocadas pelo usuário.
  • PUT /api/tarefas/:id – Atualiza o status da tarefa.
  • DELETE /api/tarefas/:id – Apaga uma tarefa.

3- Subtarefa

  • POST /api/subtarefas – Cria uma subtarefa dentro de uma tarefa.
  • GET /api/subtarefas – Lista todas as subtarefas no sistema e para qual tarefa estão atribuídas.
  • PUT /api/subtarefas/:id – Atualiza o status da subtarefa.
  • DELETE /api/subtarefas/:id – Apaga uma subtarefa.
